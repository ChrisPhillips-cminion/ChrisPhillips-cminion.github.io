Recently I have been working with IBM AppConnect Designer in Software and the connectors for LDAP and Active Directory.

There are three connectors that need to be considered here
* Microsoft Entra ID 
* Microsoft Active Directory
* LDAP Request node

<!--more-->

THe following information is available from the knowledge center and links provided but I found it useful to have all of this in one place.

Supported actions for the Microsoft Entra ID Request node:

| Objects | Actions |
|---|---|
| Devices | Create, retrieve, update, or delete devices |
| Groups  | Create, retrieve, update, or delete groups |
| Users  | Create, retrieve, update, or delete users | 
[https://www.ibm.com/docs/en/app-connect/13.0.x?topic=nodes-microsoft-entra-id-request-node](https://www.ibm.com/docs/en/app-connect/13.0.x?topic=nodes-microsoft-entra-id-request-node)


Supported actions for the Microsoft Active Directory Request node:

| Object | Actions |
|---|---|
| Computers: | Create, retrieve, update, or delete computers |
| Contacts: | Create, retrieve, update, or delete contacts |
| Entries: | Create, retrieve, update, or delete entries, add to group, remove from group, or modify distinguished name |
| Groups | Create, retrieve, update, or delete groups |
| InetOrgPerson objects | Create, retrieve, update, or delete InetOrgPerson objects |
| Organizational units |  Create, retrieve, update, or delete organizational units |
| Users | Create, retrieve, update, or delete users | 

[https://www.ibm.com/docs/en/app-connect/13.0.x?topic=nodes-microsoft-active-directory-request-node](https://www.ibm.com/docs/en/app-connect/13.0.x?topic=nodes-microsoft-active-directory-request-node)

Supported actions for the LDAP Request node:

| Object | Actions |
|---|---|
| Devices  | Create, retrieve, update, or delete devices |
| Entries  | Create, retrieve, update, or delete entries, or modify distinguished names |
| InetOrgPerson objects  | Create, retrieve, update, or delete inetOrgPerson objects |
| Organizational persons  | Create, retrieve, update, or delete organizational persons |
| Organizational units  | Create, retrieve, update, or delete organizational units |
| Persons  | Create, retrieve, update, or delete persons |
| Rooms | Create, retrieve, update, or delete rooms 

[https://www.ibm.com/docs/en/app-connect/13.0.x?topic=nodes-ldap-request-node](https://www.ibm.com/docs/en/app-connect/13.0.x?topic=nodes-ldap-request-node)
